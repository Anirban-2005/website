04 Selected Work

01
RPA · Automation

RPA-Based Social Media Automation

Problem

Manually scheduling, cross-posting and moderating social content across networks is brittle and time-expensive for creators and small teams.

Solution

A UiPath-orchestrated RPA workflow that ingests a content calendar, generates variant captions, and posts + monitors across platforms with human-in-the-loop review.

Impact

Cuts posting overhead by ~85% and standardises brand voice with rule-based captioning.

  • Python
  • Selenium
  • REST APIs

NLP · Web Scraping

Automated News Extraction

Problem

Following breaking stories across dozens of publishers is impossible manually; noise drowns the signal.

Solution

A pipeline that crawls curated sources, deduplicates via TF-IDF + cosine similarity, and summarises stories into a personal digest.

Impact

~30 minutes of daily reading condensed into a 2-minute digest.

03
DSP · AI

Automatic Radar-Based Target Detection System

Problem

Detecting and classifying moving targets in noisy radar returns requires precise DSP + ML fusion.

Solution

CFAR-based detection front-end feeding a lightweight classifier trained on synthetic + real returns; outputs range/velocity + class in real-time.

Impact

Robust detection under low SNR; suitable for embedded deployment.

  • Python

  • ML
